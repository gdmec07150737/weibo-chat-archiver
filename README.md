
# 微博聊天记录备份工具 - 本地开发指南

您当前的 Node.js 版本 (`v22.17.1`) 非常完美。

## 快速启动

1. **安装环境依赖**：
   ```bash
   npm install
   ```

2. **配置 API KEY**（可选）：
   在项目根目录创建 `.env` 文件：
   ```text
   VITE_API_KEY=你的Gemini密钥
   ```

3. **运行开发服务器**：
   ```bash
   npm run dev
   ```
   应用会自动在浏览器打开 `http://localhost:5173`。

## 为什么不能直接打开 index.html？
现代前端应用（React/Vue）依赖“模块化加载”，浏览器基于安全策略禁止本地文件（`file://`）加载这些模块。使用 `npm run dev` 会启动一个微型服务器，模拟真实的网站运行环境。

## 功能提醒
- **自动化脚本**：在“备份新记录”页面复制脚本。
- **采集过程**：请确保在浏览器中已经登录了微博网页版（api.weibo.com）。
- **数据安全**：所有备份数据均保存在您浏览器的 IndexedDB/LocalStorage 中，不会上传到任何服务器。
